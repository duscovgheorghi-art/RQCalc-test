# RQCalc

## Project Description
RQCalc is a powerful calculator application designed for quick and accurate calculations. Whether you're a student, engineer, or someone who deals with numbers regularly, RQCalc provides a user-friendly interface and a range of features to enhance your productivity.

## Features
- Supports basic arithmetic operations (addition, subtraction, multiplication, division).
- Advanced functions like exponentiation, logarithms, and trigonometric calculations.
- User-friendly interface for easy navigation.
- History log of calculations for reference.
- Customizable settings to tailor the experience.

## Installation Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/duscovgheorghi-art/RQCalc-test.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd RQCalc-test
   ```
3. **Install dependencies** (assuming you are using npm):
   ```bash
   npm install
   ```

## Configuration Guide
- Open the `config.json` file to adjust settings such as theme, precision, and history log limit.
- For help with configuration options, refer to the comments in the `config.json` file.

## Usage Examples
- **Basic Calculation**:
   ```bash
   > RQCalc 5 + 7
   12
   ```
- **Advanced Function**:
   ```bash
   > RQCalc sin(90)
   1
   ```
- **Using History**:
   ```bash
   > RQCalc history
   5 + 7 = 12
   sin(90) = 1
   ```

## Troubleshooting
- **Installation Issues**:
  - Ensure Node.js is installed on your machine. Check with `node -v`.
  - If `npm install` fails, try clearing your npm cache with `npm cache clean --force`.

- **Common Errors**:
  - If you encounter any calculation errors, check for typos in your input.
  - Refer to the history log for recent calculations to avoid repetitions.

For further assistance, feel free to open an issue in this repository or consult the documentation.
